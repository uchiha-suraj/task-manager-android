# Task Manager Android App

A Task Manager application built using:

- Kotlin
- Jetpack Compose
- Retrofit
- Room Database
- MVVM Architecture
- Repository Pattern
- UiState Pattern

## Features

- View Tasks
- Add Tasks
- Update Tasks
- Delete Tasks
- Offline Caching with Room
- API Integration using Retrofit

## Architecture

UI → ViewModel → Repository → Room/API

## Tech Stack

- Kotlin
- Jetpack Compose
- Retrofit
- Room
- Coroutines
- MVVM
